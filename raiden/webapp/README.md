# Raiden Minimal UI

A proposed Demo UI for Raiden API's

Setup
================================

### Install NPM dependencies

```
npm install
```

### Start Rest Server
Inside a python virtual Environment install flask
Navigate to the mockapi folder
```
python raidenmockapi.py
```

### Start the Python SimpleHttpServer to serve the webpage
Inside the webapp folder start the SimpleHttpServer
```
python -m SimpleHTTPServer 8080
```
You could select the port number of your choice if 8080 is occupied
